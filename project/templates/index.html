Instructions

This web app allows you to do a complete simulation of the 2022 World Cup.

You can either start with Finals Only which will allow you to run a simulation of just the 2022 FIFA World Cup tournament with the teams that are currently projected to reach the tournament.

You can also start with All Qualifiers which will allow you to simulate every conference's qualification rounds (except the OFC) starting with the round that is currently in action.

Once you run a simulation, press Home to return here.
